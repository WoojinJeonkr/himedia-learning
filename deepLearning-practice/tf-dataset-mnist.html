# 2. 이미지 전처리

# 1) 4차원으로 생성
#    - 이미지 셋 3차원: 2차원 이미지가 여러 장
#    - 이미지 픽셀마다 특성(특징)을 추출하는 처리를 하기 위한 차원 하나 더 추가
# 2) 데이터를 읽어왔을 때 픽셀마다의 값의 명암 정도만 가지고 있음(0 ~ 255)
#    - 0 ~ 1 사이로 스케일링(표준화)시켜줌
#    - 방법) 255로 나누어주면 됨
# 3) 이미지 크기는 아주 다양하므로 크기를 표준화
